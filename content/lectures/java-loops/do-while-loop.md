### Example 1:

```java
public class DoWhileLoopExample1 {
    public static void main(String[] args) {
        int i = 1;
        do {
            System.out.println("Hello World");
            i++;
        } while (i <= 5);
    }
}
```

### Example 2:

```java
public class DoWhileLoopExample2 {
    public static void main(String[] args) {
        int i = 1;
        do {
            System.out.println(i);
            i++;
        } while (i <= 10);
    }
}
```
