### Example 1:

```java
public class ForLoopExample1 {
    public static void main(String[] args) {
        for (int i = 1; i <= 5; i++) {
            System.out.println("Hello World");
        }
    }
}
```

### Example 2:

```java
public class ForLoopExample2 {
    public static void main(String[] args) {
        for (int i = 1; i <= 10; i++) {
            System.out.println(i);
        }
    }
}
```
